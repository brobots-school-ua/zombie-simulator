# Fix Plan — Краща графіка (Wishlist #2)

## Поточний стан
Вся графіка генерується кодом через `Phaser.Graphics` — прості кола, прямокутники, лінії.
Це дає "programmer art" рівень якості. Потрібно значно покращити вигляд.

## Що можна покращити

### Варіант А: Pixel Art спрайти (замінити процедурну графіку на справжні PNG)
**Плюси:** Найкращий візуальний результат, повний контроль
**Мінуси:** Потрібно малювати кожен спрайт (або генерувати через AI)
**Підхід:**
- Використати Gemini для генерації pixel art дизайнів
- Намалювати спрайти як base64 data URLs або Canvas API з більшою деталізацією
- Замінити текстури по категоріях

### Варіант Б: Покращити процедурну генерацію (поточний підхід, але краще)
**Плюси:** Не потрібні зовнішні файли, все в коді
**Мінуси:** Є обмеження того, що можна зробити через Graphics API
**Підхід:**
- Більш детальні текстури (тіні, градієнти, обводки)
- Анімації (idle, ходьба, атака)
- Ефекти частинок (кров, дим, вогонь)
- Покращене освітлення та атмосфера

### Варіант В: Комбо — процедурні текстури на Canvas + покращення ефектів
**Плюси:** Найкращий баланс якості та простоти
**Підхід:**
- Створити текстури через HTML Canvas 2D (більше можливостей ніж Phaser Graphics)
- Додати візуальні ефекти (тіні під об'єктами, частинки)
- Покращити карту (більше різноманітності)
- Покращити UI/HUD стиль

---

## Пропонований план (Варіант В — поетапно)

### Етап 1: Покращення гравця та зомбі
- Більш деталізовані спрайти (тіні, обводки, деталі одягу)
- Плавна анімація повороту (замість статичного спрайту)
- Тінь під кожним об'єктом (dark circle під ногами)
- Консультація з Gemini для кожного дизайну

### Етап 2: Покращення карти/оточення
- Більше варіантів ground тайлів (4-6 замість 3)
- Додати декоративні об'єкти: дерева, каміння, бочки, машини
- Стіни з різними текстурами (цегла, бетон, паркан)
- Плавний край зони гри (не різкий обрив)

### Етап 3: Візуальні ефекти
- Частинки крові при попаданні
- Дим/пил при русі
- Вибухові ефекти (кращі)
- Мерехтіння при попаданні (flash damage)
- Тіні від об'єктів

### Етап 4: UI/HUD
- Стильніший HUD (dark theme, прозорі панелі)
- Анімації появи тексту (damage numbers)
- Кращий шрифт/стиль для всіх текстів
- Плавніші переходи між екранами

### Етап 5: Атмосфера
- Туман/частинки в повітрі
- Зміна кольору карти з хвилями (темніше = складніше)
- Вигнеттування (темні краї екрану)
- Підсвічування при вибухах (screen flash)

---

## Черговість
Пропоную почати з **Етапу 1** (гравець + зомбі), бо це те що видно найбільше.
Потім Етап 3 (ефекти) — дає найбільше "вау" за найменше зусиль.
Потім Етап 2 (карта), Етап 4 (UI), Етап 5 (атмосфера).

Або можемо обрати щось конкретне, що тобі найбільше хочеться покращити!
