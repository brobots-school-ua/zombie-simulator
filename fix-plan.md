## Крафтові матеріали — РЕАЛІЗОВАНО ✅

- 3 типи матеріалів: дерево (wood), метал (metal), гвинти (screws)
- Випадають з зомбі при вбивстві, кожен предмет має свій окремий шанс
- Шанси випадіння:
  - Walker: дерево 20%, метал 15%, гвинти 5%
  - Runner: дерево 20%, метал 20%, гвинти 7%
  - Tank: дерево 17%, метал 17%, гвинти 10%
  - Radioactive: дерево 22%, метал 22%, гвинти 12%
  - Kamikaze: дерево 25%, метал 25%, гвинти 15%
  - Boss: всі 100% (гарантований дроп)
- Можуть випасти 0, 1, 2 або всі 3 одночасно
- Текстури: дерево (коричневе), метал (сірий з заклепками), гвинти (болт)
- UI: іконки з кількістю зліва під Score
- Змінені файли: Pickup.ts, Player.ts, Zombie.ts, GameScene.ts, UIScene.ts, BootScene.ts

## Рюкзак (Backpack) — перегляд всіх ресурсів

### Що робимо
Додаємо рюкзак — панель, яка показує крафтові матеріали:
- Дерево (Wood)
- Метал (Metal)
- Гвинти (Screws)

### Де відкривається
1. **В грі** — натискаєш **TAB** → HTML панель поверх гри
2. **В меню** — кнопка **BACKPACK** поруч з SHOP, BESTIARY (з localStorage)

### Як виглядає
```
┌─────────────────────────────┐
│  BACKPACK              [✕]  │
│─────────────────────────────│
│  Wood: 12                   │
│  Metal: 8                   │
│  Screws: 3                  │
└─────────────────────────────┘
```

### Підказка на екрані
В грі внизу (або зверху) показати текст: **"TAB — backpack"** — щоб гравець знав як відкрити рюкзак.

### Файли які змінюємо
1. **`src/scenes/UIScene.ts`** — TAB відкриває/закриває рюкзак + підказка на екрані
2. **`src/scenes/MenuScene.ts`** — кнопка BACKPACK + панель (з localStorage)

### Збереження
Матеріали зберігаються в localStorage при Game Over / виході, щоб в меню було видно запас.
