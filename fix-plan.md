# Fix Plan

## Здібності гравця (Abilities) — 3 штуки

### Загальна система
- Гравець обирає **одну здібність** перед грою (в меню, окрема кнопка ABILITIES)
- В грі: іконка обраної здібності + клавіша **F** для активації
- Без кулдауну — доступна в будь-який момент (поки що)
- Зберігається в localStorage яка здібність обрана

### Здібність 1: BIG BOMB (💣)
- Натискаєш F → біля гравця спавниться бомба (спрайт з таймером 3 сек)
- Відлік: 3...2...1... над бомбою
- Вибух: 300 damage в великому радіусі навколо гравця (radius=400)
- Боси виживають (500 HP - 300 = 200 HP)
- Візуал: величезний вибух (flash екрану + розширюється коло)

### Здібність 2: MINI NUKE (☢️)
- Натискаєш F → камера плавно відзумлюється, показуючи всю мапу
- Зомбі відмічені неоновими маркерами (колір = тип монстра) + підпис типу
- Гравець клацає на точку на мапі
- Ракета летить 5 секунд (візуально видно)
- Вибух: радіус = 500px (чверть карти від краю, mapSize=2000, 2000/4=500)
- Damage: 500 (вбиває все, боси теж)
- Після вибуху — рандомні радіоактивні плями (5-8 штук) в зоні вибуху, дамажать зомбі по 10/сек протягом 8 сек
- Камера повертається до гравця

### Здібність 3: CRYO CAPSULE (❄️)
- Натискаєш F → вся мапа заморожується (синій flash)
- Всі зомбі отримують 50 damage
- Всі зомбі заморожені на 5 секунд (не рухаються, синій tint)
- Візуал: синій overlay + частки льоду

### Нові файли
1. **`src/systems/AbilityConfig.ts`** — конфіг 3 здібностей (id, name, description, icon emoji, key)
2. **`src/systems/AbilityManager.ts`** — singleton: getSelected(), setSelected(), логіка виконання кожної здібності

### Зміни в існуючих файлах
1. **`src/scenes/BootScene.ts`** — текстури для бомби, ракети, вибухів, крижаних часток
2. **`src/scenes/MenuScene.ts`** — нова кнопка ABILITIES (квадратна, як SHOP) + панель вибору
3. **`src/scenes/GameScene.ts`** — клавіша F активує здібність, логіка бомби/нюка/заморозки
4. **`src/scenes/UIScene.ts`** — іконка здібності + підказка "F — ability"
5. **`src/entities/Zombie.ts`** — метод freeze() для заморозки

### Порядок роботи
1. AbilityConfig + AbilityManager
2. Текстури в BootScene
3. Логіка в GameScene (всі 3 здібності)
4. UI іконка + підказка
5. Панель вибору в MenuScene
6. freeze() для зомбі
